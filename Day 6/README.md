# AI_advent

## Day 6. Температура

`temperature` масштабирует распределение вероятностей токенов перед выбором следующего токена:
- **Низкая temperature** → модель почти всегда выбирает наиболее вероятный токен
- **Высокая temperature** → модель чаще выбирает менее вероятные токены
Формально: логиты делятся на `temperature` перед применением softmax.

**Интуитивно**
- **0.0 – 0.2**  
    Почти детерминированный вывод  
    Подходит для:
    - фактов
    - инструкций
    - кода
    - строгих формулировок
- **0.5 – 0.8**  
    Баланс между точностью и вариативностью  
    Подходит для:
    - объяснений
    - диалогов
    - перефразирования
- **1.0 и выше**  
    Высокая креативность, больше неожиданностей  
    Подходит для:
    - художественного текста
    - идей
    - брейншторма  
        Риск галлюцинаций выше
