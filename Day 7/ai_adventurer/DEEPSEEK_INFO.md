# DeepSeek API - Информация

## Что такое DeepSeek?

DeepSeek - это китайская компания, предоставляющая продвинутые языковые модели через API. DeepSeek API использует OpenAI-совместимый формат, что делает миграцию между сервисами очень простой.

## Преимущества DeepSeek

1. **Доступная цена** - дешевле, чем OpenAI GPT
2. **Хорошее качество** - конкурентоспособные результаты
3. **OpenAI-совместимый API** - легко переключаться между сервисами
4. **Специализированные модели** - например, DeepSeek Coder для программирования

## Доступные модели

### deepseek-chat
- **Назначение:** Общий чат и диалог
- **Применение:** Подходит для большинства задач общения
- **Контекст:** До 32K токенов

### deepseek-coder
- **Назначение:** Специализированная модель для программирования
- **Применение:** Написание кода, отладка, code review
- **Контекст:** До 16K токенов
- **Особенность:** Лучше понимает код на разных языках программирования

## Цены (актуально на 2026)

DeepSeek обычно дешевле OpenAI:
- Input: ~$0.14 за 1M токенов
- Output: ~$0.28 за 1M токенов

Точные цены смотрите на [официальном сайте DeepSeek](https://platform.deepseek.com/pricing).

## Как получить API ключ

1. Перейдите на https://platform.deepseek.com/
2. Зарегистрируйтесь (может потребоваться китайский номер телефона или email)
3. Пополните баланс (обычно минимум $5)
4. Создайте API ключ в разделе API Keys
5. Скопируйте ключ и добавьте в `RetrofitInstance.kt`

## Ограничения и особенности

- **Rate Limits:** Зависят от вашего тарифного плана
- **Доступность:** Может быть медленнее из-за географического расположения серверов
- **Языки:** Отлично работает с английским и китайским, хорошо с другими языками
- **Безопасность:** Следует тем же принципам безопасности, что и OpenAI

## Переключение обратно на OpenAI

Если вы хотите вернуться к OpenAI API:

1. В `RetrofitInstance.kt` измените:
```kotlin
private const val BASE_URL = "https://api.openai.com/"
```

2. В `ApiModels.kt` измените:
```kotlin
val model: String = "gpt-3.5-turbo" // или gpt-4
```

3. Используйте ваш OpenAI API ключ
